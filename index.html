適用於 Windows 的開源 C/C++ 整合式開發環境

Dev-C++是適用於 Windows 平台的全功能 C 和 C++ 整合開發環境 (IDE)。自1998 年第一個版本發布以來，數以百萬計的開發人員、學生和研究人員使用Dev-C++。它已出現在數十本C++ 和科學書籍中，並且仍然是全球大學和學校最喜愛的學習工具之一。
